Caf Genie

Caf Genie is an App that offers the quickest, most reliable, streamlined information about the Biola Cafeteria. No inconvenient websites, no hassle.

Caf Genie was written in the React Native JS framework as a cross-platform replacement for Miles Luders's "Caf Wizard" app. New features, such as the ability to select "Favorites" and view nutritional info, will be released soon.
